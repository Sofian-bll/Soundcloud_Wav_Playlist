Download SoundCloud playlists and convert every track to lossless WAV with preserved metadata.

Features

Full Playlist Downloads

Point it at a set URL, get every track in one run.

Lossless WAV

Conversion via ffmpeg — 1411 kbps, 44.1 kHz stereo.

Metadata Intact

Title, artist, genre, track number, date, and cover art preserved.

Quick Setup

One venv, one pip install, one ffmpeg binary. That's it.
